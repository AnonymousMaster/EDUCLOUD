¿Qué es EDUCLOUD?

Software de gran alcance, fácil de usar y de código abierto (OpenSource) de gestión de Instituciones Educativas desarrollado en Ruby on Rails y Twitter Boostrap 3.0. Es una aplicación web 2.0 que se puede instalar y acceder utilizando la computación en la nube (cloud computing ).

¿ Porqué EDUCLOUD?

  • Fácil seguimiento de los estudiantes
  • Analisis de Desempeño y asistencia
  • Generación de informes instantáneos
  • Como toda la línea de datos es y seguro, cualquier análisis se puede hacer en cualquier punto de tiempo deseado.
  • Diseño intuituvo de interfaz de usuario hace que el uso del sistema le sea más fácil para el personal de la Institucion Educativa y los estudiantes, para soporte movil y de Escritorio.
  • Altamente personalizable y fácil de personalizar
  • Construido sobre poderosas tecnologías de Ruby on Rails
  • Soporta el idioma local , y escalable a idiomas extranjeros
  • Arquitectura de Plugin, usted puede desarrollar su propio plugin de Add-On y se integran con EDUCLOUD.
  • Los informes personalizados pueden ser generados

¿Donde se puede instalar EDUCLOUD?

EDUCLOUD es una aplicación web 2.0. que se puede instalar en un servidor web remoto o un servidor web de red local (como Apache, Mongrel o Webrick ) . La arquitectura de EDUCLOUD no admite que sea instalado como una aplicación independiente. Sugerida en sistema operativo Ubuntu 13.04 / CentOS 8 o Windows 8.1 con 2 GB de RAM mínimo, 40 GB de disco duro y un procesador de nueva generación. Esto tiene que reducirse en base a las estadísticas de uso.
